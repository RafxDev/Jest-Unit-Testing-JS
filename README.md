# Laboratorio de Aprendizaje de Jest (Resuelto) (JS)

¡Bienvenido al laboratorio de testing con Jest! Aquí aprenderás los fundamentos y conceptos intermedios de las pruebas unitarias en JavaScript.

## Estructura de Lecciones

El laboratorio está estructurado en 4 niveles de dificultad:
1. **[Lección 1: Basics](file:///C:/Users/jrafa/Documents/Programacion/Jest%20Unit%20Testing%20JS/lessons/01_basics)** - Pruebas básicas y uso de `toBe()`.
2. **[Lección 2: Matchers](file:///C:/Users/jrafa/Documents/Programacion/Jest%20Unit%20Testing%20JS/lessons/02_matchers)** - Igualdad profunda, colecciones, booleanos y errores.
3. **[Lección 3: Async](file:///C:/Users/jrafa/Documents/Programacion/Jest%20Unit%20Testing%20JS/lessons/03_async)** - Promesas, `async/await`.
4. **[Lección 4: Mocks](file:///C:/Users/jrafa/Documents/Programacion/Jest%20Unit%20Testing%20JS/lessons/04_mocks)** - Mocks de funciones (`jest.fn()`), espías y ciclos de vida.

## Cómo Usar Este Laboratorio

1. **Instala las dependencias**:
   Ejecuta `npm install` en la terminal de este directorio para instalar Jest.
   
2. **Ejecuta las pruebas**:
   - Para correr todas las pruebas:
     ```bash
     npm test
     ```
   - Para correr una lección específica en modo interactivo/observador (watch):
     ```bash
     npx jest lessons/01_basics --watch
     ```

3. **Resuelve los retos**:
   Abre los archivos de prueba (`.test.js`) de cada lección. Lee las instrucciones marcadas con comentarios e implementa o arregla los fragmentos de código para que las pruebas pasen (se pongan en verde ✅).
